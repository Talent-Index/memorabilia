feat: initial commit - Memorabilia on-chain memory game

Memorabilia is a fully on-chain memory card matching game built with Dojo on Starknet, integrated with Telegram Mini Apps.

Features:
- Cairo smart contracts (game logic, leaderboard, account management)
- React frontend with beautiful animations and sound effects
- Telegram Mini App integration
- Burner accounts with session keys for gasless gameplay
- Real-time indexing with Torii
- 3 difficulty levels with score tracking

Tech Stack:
- Backend: Dojo v1.0.0-alpha.6, Cairo, Starknet
- Frontend: React 18, TypeScript, Vite, Tailwind CSS
- Integration: Telegram Mini Apps SDK
- Indexer: Torii (GraphQL)

Ready for local development and testnet deployment.
