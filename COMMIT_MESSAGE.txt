## Migration Notes

To use the Telegram bot:
1. cd telegram-bot
2. npm install
3. cp .env.example .env
4. Add BOT_TOKEN to .env
5. npm start

To test:
1. Open Telegram
2. Search @memorabilia_game_bot
3. Type /play
4. Click "Play Now" button
5. Game launches!
